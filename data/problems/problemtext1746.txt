If the value was already1, no change occurs.void unfix(int idx)Updates the value of the bit at the indexidxto0. If the value was already0, no change occurs.void flip()Flips the values of each bit in the Bitset. In other words, all bits with value0will now have value1and vice versa.boolean all()Checks if the value ofeachbit in the Bitset is1. Returnstrueif it satisfies the condition,falseotherwise.boolean one()Checks if there isat least onebit in the Bitset with value1. Returnstrueif it satisfies the condition,falseotherwise.int count()Returns thetotal numberof bits in the Bitset which have value1.String toString()Returns the current composition of the Bitset.
