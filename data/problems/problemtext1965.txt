Given anon-negativeintegernum, returntrueifnumcan be expressed as the sum of anynon-negativeinteger and its reverse, orfalseotherwise.Example 1:Input:num = 443Output:trueExplanation:172 + 271 = 443 so we return true.Example 2:Input:num = 63Output:falseExplanation:63 cannot be expressed as the sum of a non-negative integer and its reverse so we return false.Example 3:Input:num = 181Output:trueExplanation:140 + 041 = 181 so we return true. Note that when a number is reversed, there may be leading zeros.Constraints:0 <= num <= 105
