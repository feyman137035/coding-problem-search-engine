Given an array of integersnums, calculate thepivot indexof this array.Thepivot indexis the index where the sum of all the numbersstrictlyto the left of the index is equal to the sum of all the numbersstrictlyto the index's right.If the index is on the left edge of the array, then the left sum is0because there are no elements to the left. This also applies to the right edge of the array.Returntheleftmost pivot index. If no such index exists, return-1.Example 1:Input:nums = [1,7,3,6,5,6]Output:3Explanation:The pivot index is 3.Left sum = nums[0] + nums[1] + nums[2] = 1 + 7 + 3 = 11Right sum = nums[4] + nums[5] = 5 + 6 = 11Example 2:Input:nums = [1,2,3]Output:-1Explanation:There is no index that satisfies the conditions in the problem statement.Example 3:Input:nums = [2,1,-1]Output:0Explanation:The pivot index is 0.Left sum = 0 (no elements to the left of index 0)Right sum = nums[1] + nums[2] = 1 + -1 = 0Constraints:1 <= nums.length <= 104-1000 <= nums[i] <= 1000Note:This question is the same as 1991:https://leetcode.com/problems/find-the-middle-index-in-array/
